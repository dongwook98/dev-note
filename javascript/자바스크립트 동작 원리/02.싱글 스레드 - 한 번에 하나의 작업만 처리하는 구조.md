## 싱글 스레드

### 싱글 스레드

- <span style='color:skyBlue'>한 번에 하나의 작업</span>만 처리할 수 있는 구조
- JavaScript는 기본적으로 싱글 스레드 언어로, 하나의 콜 스택을 사용하여 순서대로 처리함
- 하나의 작업이 완료될 때까지 다음 작업이 실행되지 않고 대기함

### 특징

- 코드를 직관적이고 예측 가능하게 실행함
- 하나의 긴 작업이 끝나지 않으면 나머지 작업들이 모두 차단되는 단점이 있음

### 의문점

- 그런데 자바스크립트의 비동기 처리를 보면 코드가 순차적으로 실행되지 않고 예상과 다른 순서로 결과가 나타남
- 자바스크립트는 분명 <span style='color:skyBlue'>싱글 스레드</span>인데, 어떻게 이런 일이 가능할까?

- 정답은 <span style='color:skyBlue'>이벤트 루프</span>

```js
console.log('A');

setTimeout(() => {
  console.log('B');
}, 1000); // 1초 후 실행

setTimeout(() => {
  console.log('C');
}, 1000); // 1초 후 실행

console.log('D');

// 출력 결과:
// A → D → (1초 후) B, C
```
