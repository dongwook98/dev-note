## async, await 에러 처리

- 에러 처리 예시 - try, catch 사용
  - await는 Promise의 상태를 기다리고 있다가
    - 성공(fulfilled) → try 블록 실행 및 처리 결과를 반환
    - 실패(rejected) → throw를 발생 및 catch 블록 실행
  - finally 블록은 성공/실패와 상관없이 항상 실행

```js
async function fetchData() {
  try {
    // 정상 응답 or 실패 응답
    const response = await fetch('https://api.test/data');
    const data = await response.json();
    console.log(data.title);
  } catch (error) {
    console.error('Error occurred:', error);
  } finally {
    console.log('Operation complete');
  }
}

fetchData();

// 주소가 유효하다면 → try 블록에서 데이터 출력 + finally 실행
// Hello World
// Operation complete

// 주소가 잘못되었다면 → catch 블록에서 에러 출력 + finally 실행
// Error occurred: TypeError: Failed to fetch
// Operation complete
```
