## 마이크로태스크 큐(Microtask Queue)

- 일반적인 비동기 함수(예: setTimeout)의 콜백이 저장되는
  태스크 큐와는 구분됨
- 마이크로태스크 큐는 <span style='color:skyBlue'>태스크 큐보다 높은 우선순위</span>를 가지며, 현재 실행중인 코드가 종료된 직후, 이벤트 루프가 즉시 마이크로태스크 큐에 있는
  작업을 처리함

## 마이크로태스크 큐의 특징

- Promise의 .then(), .catch(), .finally() 콜백이 저장됨
- async 함수 내부의 await 이후 코드 실행도 마이크로태스크 큐에 들어감
- 태스크 큐(setTimeout, DOM 이벤트 등)보다 우선 실행됨

```js
console.log('시작');

setTimeout(() => {
  console.log('태스크 큐');
}, 0);

Promise.resolve().then(() => {
  console.log('마이크로태스크 큐');
});

console.log('끝');

// 출력
// 시작
// 끝
// 태스크 큐
// 마이크로태스크 큐 // '태스크 큐’보다 먼저 실행됨
```
