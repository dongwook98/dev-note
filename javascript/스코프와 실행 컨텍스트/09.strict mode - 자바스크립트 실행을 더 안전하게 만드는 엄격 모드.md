## strict mode란?

- ES5(2009)에서 도입된 자바스크립트의 엄격한 실행 모드
- 원래는 허용되던 "느슨한 동작"들을 금지하고 오류를 발생시킴.
- ES 모듈(import/export)과 클래스 내부는 <span style='color:skyBlue'>기본적으로 strict mode 적용</span>

- 변수 선언의 누락

  - ```js
    // 기본 모드
    x = 10; // ✅ 자동으로 전역 변수 생성됨
    ('use strict');
    x = 10; // ❌ ReferenceError
    ```

- 일반 함수의 this

  - ```js
    // 기본 모드
    function test() {
      console.log(this);
    }
    test(); // ✅ window (브라우저) / global (Node.js)

    ('use strict');
    function test() {
      console.log(this);
    }
    test(); // ❌ undefined
    ```

- 중복 파라미터

  - ```js
    // 기본 모드
    // ✅ 허용됨
    function sum(a, a, b) {
      return a + b;
    }

    ('use strict');
    // ❌ SyntaxError
    function sum(a, a, b) {}
    ```
