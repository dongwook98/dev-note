## this 키워드와 명시적 바인딩

- 명시적 바인딩

  - 기본적으로 this는 호출 방식에 따라 결정되지만, call, apply, bind를 사용하면 명시적으로 특정 객체에 this를 바인딩할 수 있음

- call

  - 함수를 즉시 실행하면서 this를 명시적으로 지정
  - 첫번째 인자로 this로 사용할 객체를 전달하고, 이후의 인자는 함수의 인수로 전달됨

- apply

  - call과 동일하지만, 함수의 인자를 배열로 전달

- bind
  - 함수를 즉시 실행하지 않고, this가 고정된 새로운 함수를 반환
  - 이후에 해당 함수를 실행하면, 항상 특정 객체를 this로 사용

## 명시적 바인딩 예시 코드

```js
function introduce(language, country) {
  console.log(`${this.name} is learning ${language} in $
{country}.`);
}

const person = { name: 'Alice' };

// call: 개별 인자로 전달하며 즉시 실행
introduce.call(person, 'JavaScript', 'USA');
// "Alice is learning JavaScript in USA."

// apply: 배열로 인자를 전달하며 즉시 실행
introduce.apply(person, ['Python', 'Canada']);
// "Alice is learning Python in Canada."

// bind: 새로운 함수를 반환하며, 이후 실행 가능
const boundIntroduce = introduce.bind(person, 'Java', 'UK');
boundIntroduce();
// "Alice is learning Java in UK."
```
