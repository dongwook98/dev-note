## this 키워드와 명시적 바인딩

### this 키워드

- 현재 실행 중인 함수가 속한 <span style="color:skyBlue">실행 컨텍스트(객체)</span>를 참조하는 키워드
- 함수의 호출 방식에 따라 this의 값이 결정됨

### this 동작 방식

- #### 전역 컨텍스트에서 this

  - 브라우저에서는 window, Node.js에서는 global 객체를 가리킴

  - ```js
    // window (브라우저) / global (Node.js)
    console.log(this);
    ```

- #### 객체의 메서드 내부에서의 this

  - 호출된 객체를 가르킴

  - ```js
    const obj = {
      name: 'Alice',
      greet() {
        console.log(this.name);
      },
    };

    obj.greet(); // "Alice"
    ```

- #### 일반 함수에서의 this

  - strict mode에서는 undefined, 그 외에는 전역 객체(window/global)를 가리킴

  - ```js
    function showThis() {
      console.log(this);
    }

    showThis();
    // window (브라우저) / global (Node.js)
    // strict mode에서는 undefined
    ```

> strict mode: ES5에서 도입된 엄격 모드, 실수를 줄이고 코드 오류를 더 엄격하게 검사하는 실행 방식
