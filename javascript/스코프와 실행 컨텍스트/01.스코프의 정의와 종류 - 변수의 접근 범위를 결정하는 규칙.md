## 스코프의 정의와 종류

### 스코프(Scope)

- **변수나 함수가 접근할 수 있는 범위를 의미**
- JavaScript에서는 변수가 선언된 위치에 따라 접근 가능한 범위가 달라지며, 이를 **스코프**라고 부름

### 전역 스코프

- 코드 어디서든 접근 가능한 범위
- 전역에서 선언된 변수는 모든 곳에서 사용 가능

### 함수 레벨 스코프

- 함수 내부에서 선언된 변수는 오직 해당 함수 내부에서만 접근 가능
- `var` 키워드는 함수 스코프를 따름 (**var는 블록 레벨 스코프를 지원안함**)

### 블록 레벨 스코프

- 중괄호 `{}`로 감싸진 블록 내부에서만 변수에 접근 가능
- `let`과 `const`는 블록 스코프를 따름

### 정리

- `let`과 `const`는 `var`보다 훨씬 세밀하게 스코프를 관리 가능
- `var`는 함수 단위까지만 구분이 가능했지만, `let`과 `const`는 함수는 물론이고 그 안의 작은 블록 단위까지도 구분해서 변수를 관리 가능 -> 훨씬 직관적이고 안전하게 코드 작성 가능

```js
// 1. 전역 스코프
var globalVar = 'I am global';
console.log(globalVar); // 가능

// 2. 함수 레벨 스코프
function funcScope() {
  var funcVar = 'I am function scoped';
}
console.log(funcVar); // 에러 (함수 내부에서만 접근 가능)

// 3. 블록 레벨 스코프
if (true) {
  let blockVar = 'I am block scoped';
  console.log(blockVar); // I am block scoped
}
console.log(blockVar); // 에러 (블록 내부에서만 접근 가능)

// 추가로 알아둘 점: var는 블록 스코프를 무시
function test() {
  if (true) {
    var x = 10;
  }
  console.log(x); // 10 (블록을 벗어나도 접근 가능)
}
test();
```
