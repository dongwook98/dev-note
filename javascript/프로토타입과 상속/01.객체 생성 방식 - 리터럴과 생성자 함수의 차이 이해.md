## 객체를 생성하는 두 가지 방식

### 객체 리터럴 방식

- 가장 간단한 객체 생성 방법으로, 중괄호 {}를 사용하여 객체를 정의함
- 속성과 값을 즉시 기술하고 필요 시 메서드도 함께 정의할 수 있음

```js
const person = {
  name: 'Alice',
  age: 25,
  greet: function () {
    console.log(`Hello, my name is ${this.name}`);
  },
};

person.greet(); // "Hello, my name is Alice"
```

### 생성자 함수 방식

- new 키워드를 사용하여 동일한 구조의 여러 객체를 쉽게 생성할 수 있음
- this 키워드를 사용하여 동적으로 속성을 할당함
- new를 생략하면 일반 함수 규칙이 적용되어 this가 전역 객체(window/global) 또는 undefined(strict mode)로 바인딩됨 -> 의도와 다르게 동작하므로 반드시 new와 함께 사용해야 함

```js
function Person(name, age) {
  this.name = name;
  this.age = age;
  this.greet = function () {
    console.log(`Hello, my name is ${this.name}`);
  };
}

const person1 = new Person('Alice', 25);
const person2 = new Person('Bob', 30);

person1.greet(); // "Hello, my name is Alice"
person2.greet(); // "Hello, my name is Bob"
```
