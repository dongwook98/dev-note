## 1. 중첩 객체(Nested Object)

- 객체 속성 안에 또 다른 객체가 포함될 수 있음

- TS에서는 각 속성의 타입을 명시 가능

### 예제: 자바스크립트 객체

```ts
const product = {
  id: 'abc1',
  price: 12.99,
  tags: ['great-offer', 'hot-and-new'],
  details: {
    title: 'Red Carpet',
    description: 'A great carpet - almost brand-new!',
  },
};
```

### 2. 타입스크립트에서 타입 정의

```ts
const product: {
  id: string;
  price: number;
  tags: string[];
  details: {
    title: string;
    description: string;
  };
} = {
  id: 'abc1',
  price: 12.99,
  tags: ['great-offer', 'hot-and-new'],
  details: {
    title: 'Red Carpet',
    description: 'A great carpet - almost brand-new!',
  },
};
```

### 포인트

- details 속성도 객체이므로, 내부 속성 `title`과 `description`에 타입 지정

- 배열 타입도 `string[]`처럼 지정 가능

- TS는 중첩 객체도 타입 검사를 수행하여 안전성을 확보

### 3. 핵심 정리

- 객체 타입은 **속성 단위로 타입 지정 가능**

- 객체 속성 안에 또 다른 객체가 들어가도 타입스크립트가 **중첩 구조까지 추론 및 검사**

- 복잡한 구조에서도 **타입 안전성 확보** 가능
