## 타입스크립트 핵심 타입 심화

### 1. number 타입

- 자바스크립트와 타입스크립트 모두 모든 숫자는 float로 취급

- 정수와 실수를 구분하지 않음

- 예시: 5와 5.0은 동일하게 처리됨

```ts
const number1: number = 5;
const number2: number = 2.8;
console.log(number1 + number2); // 7.8
```

### 2. string 타입

- 텍스트를 나타내는 타입

- 작은따옴표 `''`, 큰따옴표 `""`, 백틱 <code>``</code> 모두 사용 가능

- 백틱은 템플릿 리터럴로 문자열 내 데이터 삽입 가능

```ts
const name: string = 'TypeScript';
const greeting: string = `Hello, ${name}!`;
console.log(greeting); // Hello, TypeScript!
```

### 3. boolean 타입

- 참(true) / 거짓(false) 값을 표현

- 조건문(if)에서 주로 사용

```ts
const isActive: boolean = true;

if (isActive) {
  console.log('활성 상태입니다.');
}
```

### 4. number와 boolean을 활용한 함수 예시

```ts
function add(n1: number, n2: number, showResult: boolean, phrase: string) {
  const result = n1 + n2; // 숫자 계산 후
  if (showResult) {
    console.log(phrase + result); // 문자열과 결합하여 출력
  } else {
    return result;
  }
}

const number1 = 5;
const number2 = 2.8;
const printResult = true;
const resultPhrase = 'Result is: ';

add(number1, number2, printResult, resultPhrase);
// 출력: Result is: 7.8
```

- showResult가 true이면 함수 내부에서 바로 출력

- showResult가 false이면 결과값을 반환

- phrase를 문자열로 전달하여 출력 커스터마이징 가능

- 숫자 계산과 문자열 결합을 분리하여 타입 안전성 확보

### 5. 핵심 요약

- number: 모든 숫자 동일하게 처리, 정수/실수 구분 없음

- string: 텍스트 표현, 템플릿 리터럴로 동적 문자열 생성 가능

- boolean: 참/거짓 값, 조건문과 로직 제어에 사용

- TS는 **타입을 명시적으로 할당**하여 개발 중 오류 예방

- 함수에서 타입을 지정하면 **런타임 전에 잘못된 입력 차단 가능**
