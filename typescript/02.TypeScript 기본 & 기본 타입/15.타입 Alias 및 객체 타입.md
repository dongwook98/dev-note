# 타입 알리어스(Type Alias)와 객체 타입

타입 알리어스를 사용하면 **복잡한 객체 타입에도 이름을 붙여 재사용**할 수 있습니다.  
단순히 유니언 타입만 저장하는 것이 아니라, **객체, 배열, 리터럴 등 모든 타입**에 적용 가능합니다.

## 1. 객체 타입에 타입 알리어스 적용

```ts
type User = { name: string; age: number };

const u1: User = { name: 'Max', age: 30 }; // 유효
```

- `User`라는 이름으로 객체 구조를 정의

- 반복적인 타입 작성 없이 여러 곳에서 재사용 가능

## 2. 타입 알리어스로 함수 매개변수 단순화

### 원래 코드

```ts
function greet(user: { name: string; age: number }) {
  console.log('Hi, I am ' + user.name);
}

function isOlder(user: { name: string; age: number }, checkAge: number) {
  return checkAge > user.age;
}
```

### 타입 알리어스 적용

```ts
type User = { name: string; age: number };

function greet(user: User) {
  console.log('Hi, I am ' + user.name);
}

function isOlder(user: User, checkAge: number) {
  return checkAge > user.age;
}
```

- 객체 타입을 한 곳에서 정의하고 재사용

- **중복 제거** 및 **유지보수 용이**

- 타입 변경 시 한 곳만 수정하면 전체 코드에 적용 가능

## 3. 핵심 포인트

- 타입 알리어스는 **객체 타입, 배열, 유니언, 리터럴 등 모든 타입**에 사용 가능

- 반복적인 타입 선언을 줄이고, 코드의 가독성과 유지보수성을 향상

- 함수, 변수, 상수 등 여러 곳에서 동일 타입을 재사용할 수 있음
