## 1. 배열(Array) 기본

- 자바스크립트와 마찬가지로 배열은 여러 값의 집합

- 숫자, 문자열, 불린, 객체 등 다양한 타입 사용 가능

- 중첩 배열도 가능

- 혼합 타입 배열도 만들 수 있지만, TS에서는 명시적으로 타입 지정 필요

```ts
const hobbies = ['Sports', 'Cooking']; // 문자열 배열
```

## 2. 배열 타입 지정 방법

### (1) 타입 추론

- 배열 선언 시 초기값을 넣으면 TS가 타입을 추론

```ts
const hobbies = ['Sports', 'Cooking']; // TS 추론: string[]
```

### (2) 명시적 타입 지정

```ts
let favoriteActivities: string[] = []; // 빈 배열, 문자열만 가능
favoriteActivities.push('Reading'); // OK
favoriteActivities.push(5); // 오류: number는 string[]에 할당 불가
```

### (3) any 사용

- 어떤 타입이든 허용하지만 TS 장점 상실

```ts
let mixedArray: any[] = ['Hello', 42, true];
```

## 3. 배열 순회

- `for...of` 문으로 배열 순회 가능

- 배열 요소의 타입을 TS가 자동으로 추론

```ts
for (const hobby of hobbies) {
  console.log(hobby.toUpperCase()); // string이므로 문자열 메서드 가능
}
```

- 타입 추론 덕분에 잘못된 메서드 호출 시 즉시 오류

```ts
hobby.map(); // 오류: string에는 map 메서드 없음
```

## 4. 핵심 포인트

- 배열 선언 시 타입을 명시하거나 초기값으로 추론 가능

- TS는 배열 요소 타입을 추적하여 안전하게 메서드 사용 가능

- any를 쓰면 유연하지만 타입 안전성 포기

- 중첩 배열과 객체 배열도 동일한 방식으로 처리 가능
