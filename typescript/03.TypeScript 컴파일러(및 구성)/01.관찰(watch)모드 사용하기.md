## TypeScript 감시 모드(Watch Mode) 핵심 정리

- TypeScript 파일을 수정할 때마다 수동으로 tsc 파일명 명령을 반복 실행할 필요가 없음

- `--watch` 또는 `-w` 옵션을 사용하면 파일 변경 시 자동으로 재컴파일

```bash
tsc app.ts --watch

# 또는

tsc app.ts -w
```

- 파일을 저장하면 즉시 다시 컴파일되고 `dist` 폴더(output 파일)에 자동 반영됨

- 브라우저가 자동으로 새로고침(npm start나 dev 서버 사용 시)되므로 개발 흐름이 빨라짐

- 타입 오류가 발생하면 자동 컴파일 단계에서 바로 표시되어 즉각 확인 가능

- 단점: **여전히 특정 파일을 지정해야 한다는 부담이 있음**

  - (즉, 더 큰 프로젝트에서는 보통 tsconfig를 활용)
